MATLAB Graph Editorの使い方　byマハト

MATLAB Graph EditorはMATLABのfigureを使用してグラフィカルに
グラフを生成するツールです．


1.起動の仕方
　まずMATLABのカレントディレクトリをgraph_editorが入っているフォルダに移動します．

　[1]新規作成の場合
	コマンドウィンドウに「graph_editor(ノードの数)」と入力する．

　[2]グラフラプラシアンや隣接行列からグラフを生成する場合
	ワークスペースにグラフラプラシアンまたは隣接行列を入れた変数(ここではAとする)
	を作り，コマンドウィンドウに「graph_editor(A)」と入力する．

2.主な機能
　[1]エッジの生成
　　(1)エッジの始点となるノードをクリックします．この時ノードの輪郭が点線になります．
　　(2)次にエッジの終点となるノードをクリックします．始点と終点の間にエッジが生成されます．
　　(3)「Weight on」モード(下記参照)の場合は，エッジ生成後にそのエッジの重み編集ができます．
　　　重み編集を終わるには「Esc」もしくはfigure上の重み以外の部分をクリックします．
　　(4)始点をクリックした状態からエッジ生成をキャンセルしたい場合は，始点もしくはfigure上の　　　ノード以外の部分をクリックします．

　[2]ノード，エッジ，ノード名，重みの設定変更
　　各オブジェクトで右クリックするとメニューが表示され，そこから色や大きさ，太さ等を
　　変更することができます．

3.figureメニューによる操作
　[1]Open
　　Graph Editorで過去に作成したグラフの.figファイルを開きます．
　　※MATLABから直接.figファイルを開くと再編集できません．必ずこのメニューを使って
　　　.figファイルを開いてください．

　[2]Save / Save as
　　生成したグラフを好きなファイル形式で保存します．

　[3]Save Option
     「Clear BackColor」にチェックを付けることで，グラフを画像ファイルとして保存する時に
　　背景を透明にすることができます．

　[4]Export Matrix
　　生成したグラフラプラシアンまたは隣接行列を.matファイルとして保存します．.matファイルの
　　変数名は，グラフラプラシアンの場合は「gl」，隣接行列の場合は「ad」となっています．

　[5]Weight on
　　「Weight on」にチェックを付けると，エッジに重みを設定できるようになります．重みの
　　初期値は全て1となっています．

　[6]All Node setting等
　　figure上のすべてのノードやエッジ等の設定を一括で変更します．

　[7]Axes on / grid on
　　チェックを付けることでfigure上に座標軸やグリッドを表示します．
　　画像ファイルに一緒に保存されてしまうため，保存する際は非表示にすることをお勧めします．

　[8]Display
　　生成したグラフのグラフラプラシアンや隣接行列をMATLABのコマンドウィンドウに
　　MATLAB形式で表示します．

4.マウスを使った特殊な操作
　[1]ノードの移動
　　「SHIFT」キーを押しながらノードをドラッグすることで，ノードを好きな位置に移動できます．

　[2]ノードのサイズ変更
　　ノードをクリックし，輪郭が点線になっている状態でマウスのスクロールホイールを回すと
　　ノードのサイズを変更することができます．

　[3]ノード名や重みの編集
　　ノード名や重みをダブルクリックすることで，編集モードに入ることができます．
　　編集を終わるには「Esc」またはfigure上の編集テキスト以外の部分をクリックします．

　[4]ノード名や重みの移動
　　ノード名や重みをドラッグすることで好きな位置に移動できます．

　[5]ノード名のフォントサイズの変更
　　ノード名をクリックし，四角い枠が表示された状態でマウスのスクロールホイールを回すと
　　フォントサイズを変更することができます．
